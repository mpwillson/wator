USAGE:

Start: Start or continue simulation.
Stop: Pause simulation.
Reset: Clear ocean and reseed.

Alter the values for Breeding Age and Starvation to change the behaviour of the simulator.

Wator is an implemention, in Clojurescript, of the predator/prey simulation described by A.K. Dewdney in "The Armchair Universe". Source code can be found on Github.
